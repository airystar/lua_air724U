setAuthApn ([prot=0][, apn=""][, user=""][, pwd=""]) 
设置专网卡APN(注意：在main.lua中，尽可能靠前的位置调用此接口)
 第一次设置成功之后，软件会自动重启，因为重启后才能生效

Parameters:

  • prot number [此参数可选，默认值为： 0] 
    加密方式， 0:不加密  1:PAP  2:CHAP
  • apn string [此参数可选，默认值为： ""] 
    apn名称
  • user string [此参数可选，默认值为： ""] 
    apn用户名
  • pwd string [此参数可选，默认值为： ""] 
    apn密码

Returns:

  • nil

Usage:

  • c = link.setAuthApn(2,"MYAPN","MYNAME","MYPASSWORD")
