Module aLiYunOta

 目前固件签名算法仅支持MD5

Functions

setVer (version) 
设置当前的固件版本号

Parameters:

  • version string 
    当前固件版本号

Returns:

  • nil

Usage:

  • aLiYunOta.setVer("MCU_VERSION_1.0.0")
setName (name) 
设置新固件保存的文件名

Parameters:

  • name string 
    新固件下载后保存的文件名；注意此文件名并不是保存的完整路径，完整路径通过setCb设置的回调函数去获取

Returns:

  • nil

Usage:

  • aLiYunOta.setName("MCU_FIRMWARE.bin")
setCb (cbFnc) 
设置新固件下载后的回调函数

Parameters:

  • cbFnc function 
    新固件下载后的回调函数
 回调函数的调用形式为：cbFnc(result,filePath)，result为下载结果，true表示成功，false或者nil表示失败；filePath为新固件文件保存的完整路径

Returns:

  • nil

Usage:

  • aLiYunOta.setCb(cbFnc)
